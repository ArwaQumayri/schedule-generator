# Schedule Generator (AI Mini Project)

An AI-based tool that generates conflict‑free university schedules using Backtracking and Constraint Satisfaction. The system reads a CSV file of course sections and produces all valid combinations, with optional support for selecting a preferred day off.

## Features
- Generates all non-conflicting schedules
- Detects time and day overlaps
- Supports optional day-off constraint
- Uses Backtracking Search for efficient pruning
- Simple CSV-based input

## How It Works
1. Load the CSV file containing course groups.
2. Enter the desired course codes.
3. (Optional) Choose a day off.
4. The algorithm filters groups, checks constraints, and builds valid schedules.
